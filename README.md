# Project Chrysalis: The Architect's Swarm

A game about systems engineering, emergent swarm behavior, and distributed systems survival.

## Overview
You are the Architect, stationed in orbit above Kepler-452b. You do not pilot drones; you write the behavioral protocols that govern a massive, fragile swarm of micro-drones.

## Foundational Documents
- [**Strategy & Vision**](./STRATEGY.md): Core loop, design pillars, and key mechanics.
- [**Narrative North Star**](./NARRATIVE.md): The Act I-IV progression and the evolution of the Architect.

## Technical Foundation
This project utilizes a custom Go-based scripting language (**P-Script**) and a deterministic simulation engine to manage thousands of autonomous units.
