# Chrysalis Swarm Client

> **Role**: Visual Presentation & Input (The "Eyes")

This directory contains the Godot 4 project. It is responsible for:
- Rendering the simulation state received from the Go core.
- Handling user input and UI.
- Managing the visual assets and scene trees.

## Key Files
- `main.tscn`: The primary gameplay viewport.
- `main.gd`: The bridge script that communicates with the Go process.
