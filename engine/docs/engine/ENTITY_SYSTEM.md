# Game Engine — Entity-Component-System (ECS)

Organizes gameplay objects.

## ECS Structure
- **Entities**: Unique identifiers representing simulation assets.
- **Components**: Plain data structs (Position, NetworkSync, RigidBody).
- **Systems**: Enforces rules (MovementSystem, CollisionSystem).
