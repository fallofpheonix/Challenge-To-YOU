# Metrics

## Primary Metrics

- Curiosity Rate (CR)
- Experiment Rate (ER)
- Explanation Rate (XR)
- Transfer Rate (TR)
- Story Rate (SR)
- Time To First Voluntary Experiment (TTFVE)
- Hypothesis Count

## Collection Notes

Record exact player quotes, timestamps, and what the player did immediately before each quote.

## Observation Fields

- Tester name or ID
- Start time
- End time
- Number of voluntary experiments
- Number of required experiments
- Correct explanation quality
- Moments of confusion
- Moments of delight
- Notable quotes
- Count of voluntary versus required actions
- Whether the player asked to try another variation
- Whether the player could explain the result without help
