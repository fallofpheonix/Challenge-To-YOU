# Prototypes

Each prototype is self-contained so evidence, notes, media, and results stay together.

## Maturity Levels

- P0: Thought experiment
- P1: Paper prototype
- P2: Interactive prototype
- P3: Playtest prototype
- P4: Production candidate

## Folder Convention

Each prototype folder may include:

- README.md
- hypothesis.md
- metrics.md
- results.md
- notes.md
- build/
- video/
- screenshots/
