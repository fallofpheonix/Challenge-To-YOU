## 1. Medieval Magitech
- **Theme**: Runic Magic and Leyline Networks.
- **Reality Layer**: `Leyline Fabric`.
- **Player Role**: `Rune Binder`.
- **Programming Discipline**: `Axiomatic Programming`. Runes are logical assertions and statements. Combining opposite forces (e.g. fire and ice) creates explosive state changes.
- **Visual Identity**: Glowing stone walls, parchment scroll indicators, golden runic pathways, emerald ley-energy indicators.
- **Music Style**: Gregorian-chant drone with medieval flutes and low double-bass pulses.
- **Dungeons**: `Leyline Cathedral`, `High Sanctum Focal Loop`.
- **Unique Mechanics**: `Rune Combination`. Script lines assert rune bindings that mutate state maps.

## 7. Quantum Nexus
- **Theme**: Quantum computing and probability manipulation.
- **Reality Layer**: `Quantum Gates Matrix`.
- **Player Role**: `Quantum Architect`.
- **Core Concepts**: Superposition, Entanglement, Qubits, Grover's Algorithm, Shor's Algorithm, Teleportation, Measurement Collapse.
- **Programming Discipline**: `Circuit Optimization`. Applying gates (Hadamard, CNOT, Pauli-X) to qubits to achieve targeted probability amplitudes before collapse.
- **Visual Identity**: Holographic green probabilities, shifting wave-functions, crystalline matrices, split particle trails.
- **Music Style**: Shimmering bell pads, microtonal synth chords, high-frequency digital sparkles.
- **NPCs**: `Amplituda (Choir Priestess)`, `Dr. Dirac (Defector)`.
- **Cities**: `EPR Station`, `Hilbert City`.
- **Landmarks**: `The Hadamard Bridge`, `Superposition Dome`.
- **Dungeons**: `Shor's Ledger Vault`, `Grover's Search Engine`.
- **Challenge Categories**: Amplitude Tuning, Entanglement Stitching, Circuit Minimization.
- **Collectibles**: `Entangled Qubit Cores`, `Phase Seals`.
- **Hidden Secrets**: Injecting a custom phase shift that forces the gate to resolve to an unscripted state.
- **Achievements**: `Confluence of Probabilities` (Achieve 100% target probability with < 4 gates).
- **Environmental Hazards**: `Phase Decoherence Storms`. Destroys qubit states if not stabilized within 3 cycles.
- **Unique Puzzle Mechanics**: `Measurement Trap`. Activating a gate without stabilizing phase alignment collapses the level state, causing an instant purge.

## 14. Origin Kernel
- **Theme**: Low-level operating system architectures and birth of reality.
- **Reality Layer**: `Universal Kernel`.
- **Player Role**: `Kernel Ascendant`.
- **Core Concepts**: Operating Systems, Kernel, Schedulers, Interrupts, Memory, Processes, System Calls, Device Drivers, Boot Sequence.
- **Programming Discipline**: `Interrupt Handling`. Writing low-level system-call interrupt handlers that allocate CPU slices to reality processes.
- **Visual Identity**: Pure gold circuit lines, black terminal grids, glowing process stacks, BIOS start sequences.
- **Music Style**: Orchestral cybernetic hymn, synth-choirs, massive organ chords, static white noise swells.
- **NPCs**: `The Bootstrap Node`, `BIOS (The First Voice)`.
- **Cities**: `The Bootstrap Sector`, `Kernel Center`.
- **Landmarks**: `The Main Process Stack`, `The Interrupt Gate`.
- **Dungeons**: `The First Compiler Vault`, `Memory Core Zero`.
- **Challenge Categories**: Interrupt Servicing, Page Table Mapping, Boot Sequence Assembly.
- **Collectibles**: `Kernel Keys`, `Golden Process Chips`.
- **Hidden Secrets**: Triggering a system interrupt 0x00 (division by zero) to force a cold reboot of the universe.
- **Achievements**: `Kernel Ascendance` (Successfully execute the Boot Sequence without memory faults).
- **Environmental Hazards**: `Kernel Panic`. Instantly crashes the WebSocket session, resetting the level.
- **Unique Puzzle Mechanics**: `CPU Cycle Management`. You have a fixed pool of 100 cycles per turn. Every instruction costs cycles. The player must schedule tasks optimally.
