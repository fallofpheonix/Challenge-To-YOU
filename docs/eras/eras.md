## 1. Difficulty & Level Progression

### Era I: Logic & Systems (Beginner to Intermediate)
- **Universes**: Medieval Magitech, Cyberpunk Neon, Cosmic Void, Silicon Wastes, Neural Labyrinth.
- **Focus**: Syntax, basic control flow, variable bindings, single-process logic, and ML prompting.
- **Vigilance Scaling**: Slowly builds up. Maximum operations budget of 5,000 steps per script execution.

### Era II: Distributed Paradigms (Intermediate to Advanced)
- **Universes**: Chrono Registry, Quantum Nexus, BioForge Genome, Data Abyss, Cloud Dominion.
- **Focus**: Version control rebasing, quantum logic gates, CRISPR edits, database transactions, cloud scaling.
- **Vigilance Scaling**: Rapid spikes upon syntax failures. Operations budget is limited to 2,500 steps.

### Era III: Bare-Metal Core (Advanced to Master)
- **Universes**: Machine Cathedral, Cipher Realm, Fractal Dream, The Kernel Beyond.
- **Focus**: Assembly pipelines, cryptography decrypts, fragment shaders, kernel schedulers, memory pagination.
- **Vigilance Scaling**: Instant alerts upon compilation faults. Operations budget is limited to 1,000 steps.

## 2. Global Achievements (Mastery Milestones)

- **`Runic Scholar`**: Complete all Medieval Magitech challenges.
- **`Concurrency Champion`**: Resolve 10 race conditions without mutex deadlocks.
- **`Compiler Ascendant`**: Optimize an AST program to compile with zero dead branches.
- **`Bare-Metal Scavenger`**: Solve 20 Silicon Wastes challenges using only register R0-R3.
- **`Adversarial Specialist`**: Infiltrate Cortex using a jailbreak payload of under 20 characters.
- **`Timeline Restorer`**: Rebase 10 broken Chrono branches without generating merge conflict tags.
- **`Quantum Architect`**: Stabilize 5 Grover search circuits with 100% target probability.
- **`Genome Splicer`**: Splice 15 DNA sequences without triggering cellular mutations.
- **`ACID Purist`**: Complete the Data Abyss campaign with zero transaction rollback errors.
- **`Cluster Overlord`**: Route 100,000 mock DDoS packets around container clusters without an OOM reboot.
- **`Cycle Smith`**: Schedule an assembly pipeline to run with zero data-forwarding stalls.
- **`Root Revoker`**: Break 15 cipher locks using padding-oracle scripts.
- **`Shader Painter`**: Render a custom PBR specular light portal with 60 FPS refresh parameters.
- **`Kernel Walker`**: service 50 hardware interrupts without triggering a Kernel Panic.
- **`Multiversal Ascendant`**: Complete all 560 challenges across the 14 universes.

---

## 3. Endgame: The Infinite Mainframe

Upon defeating the *First Process*, players unlock the **Infinite Mainframe**:
- **Procedural Challenges**: The engine procedurally builds level challenges combining multiple universe rules (e.g. write an assembly program that interfaces with a database query via a quantum circuit).
- **Leaderboards**: Cycles spent and memory footprint rankings for all players worldwide.
- **Mastery Trials**: Boss rushes where players must override all 14 bosses sequentially under a total cycle limit.
