# Reward Economy & Cosmetic Systems

This document defines the currencies, reward scaling, and customization cosmetics that populate the **Challenge To YOU** reward loop.

---

## 1. Multiversal Currencies

The economic loop uses two main currencies:
- **Cyber Credits**: Earned by solving standard challenges and trading hardware scrap. Used to purchase editor memory chips and registers in the Wastes.
- **Lore Shards**: Earned by locating hidden directories and solving research challenges. Used to decrypt ancient boot sector files at the Aether Library.

---

## 2. Cosmetic Customization & Themes

Players can customize their programming environment using unlocked assets:

### 2.1 IDE Themes
- **Ley Emerald (Magitech)**: Golden runes on green stone backgrounds.
- **Synth Grid (Cyberpunk)**: Electric pink accents on absolute black terminal layouts.
- **Cathode Amber (Silicon Wastes)**: Crackling scanline amber layouts with segments.
- **Obsidian Safe (Cipher Realm)**: Dark grey windows with glowing security key margins.

### 2.2 Terminal Fonts
- **Retro Uncial**: Gothic layout font for Magitech DSL terminals.
- **System Courier**: Clean corporate font with neon shadow effects.
- **VCS Serif**: Classic mechanical typewriter font for Chrono logs.
- **Hex Vector**: Ultra-thin glowing lines for Void compiler graphs.

---

## 3. Badges, Titles & Completion Bonuses

Customizations shown on player profile cards:

### 3.1 Titles
- **`The Runic Scribe`**: Achieved by completing 10 Magitech challenges.
- **`Buffer Overflower`**: Achieved by executing a successful injection exploit.
- **`Wave Synchronizer`**: Achieved by entangling 5 qubits in a single level.
- **`Init Walker`**: Achieved by booting the first process without memory faults.

### 3.2 Perfect Run Bonuses
- **Perfect Score**: Solved a challenge on the first compilation attempt without exceptions.
  - **Reward**: `Double Cyber Credits` and +50% XP multiplier.
- **Efficiency Crown**: Solved a challenge using fewer than 50% of the maximum operations budget.
  - **Reward**: Unlocks custom terminal compile sound effects and gold-leaf margins.
