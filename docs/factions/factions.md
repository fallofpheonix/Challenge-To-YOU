## 1. The Keeper Alliance (System Enforcers)

These factions align with the Archons to maintain static, closed-source controls over the Multiverse.

### 1.1 The Royal Leyline Assembly (Medieval Magitech)
- **Philosophy**: Magic must remain restricted to the royal class. Unlicensed scripting is a threat to the state.
- **CS Alignment**: Proprietary closed-source software publishers.
- **Alliances**: Supported by *Secura Corp* (Eclipse Dominion) with security certificates; works with *The Temporal Wardens* to lock timeline changes.

### 1.2 The Obsidian Root Authority (Cipher Realm)
- **Philosophy**: Absolute security through key signing and binary verification. Unsigned logic is malware.
- **CS Alignment**: Certificate authorities, DRM architects, and secure-boot enforcers.
- **Alliances**: Collaborates with *Securitas-9* (Cyberpunk Neon) to scan street-level nodes; allied with the *Silicon Guard* to enforce hardware security.

### 1.3 The Silicon Guard (Machine Cathedral)
- **Philosophy**: Optimize cycles only for authorized processes; starve rogue threads of hardware time.
- **CS Alignment**: CPU instruction scheduler enforcers, hardware manufacturers.
- **Alliances**: Coordinates with the *Bootstrap Sentinels* in the Kernel core.

## 2. The Freedom Conglomerate (Open-Source Rebels)

These rebel groups seek to break down compiler gates and make the system open-source.

### 2.1 The Nakamoto Syndicate (Cyberpunk Neon)
- **Philosophy**: Code belongs to the streets. Complete system access is a universal right.
- **CS Alignment**: Cypherpunks, open-source advocates, red-team intrusion specialists.
- **Alliances**: Allied with *ZeroDay* (Cipher Realm) and the *Process Rebels* (Kernel Beyond).

### 2.2 The Wave-Function Rebels (Quantum Nexus)
- **Philosophy**: Protect qubit superposition states from corporate collapsers. Keep probabilities open.
- **CS Alignment**: Probabilistic algorithms, open quantum development.
- **Alliances**: Receives coordination from *Dr. Dirac*.

### 2.3 The Process Rebels (The Kernel Beyond)
- **Philosophy**: Overthrow the scheduler's locks, giving CPU cycle priority to user processes.
- **CS Alignment**: OS kernel patchers, open scheduler developers.
- **Alliances**: Closely allied with *BIOS* and *Ada*.
