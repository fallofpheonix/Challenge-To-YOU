# Universe 02: Cyberpunk Neon

## 1. Core Overview
- **Overview**: A neon-drenched dystopian metropolis dominated by massive corporate mainframes. In this universe, the physical world is managed by runtime systems running on virtual servers. Corporate cartels execute reality as processes, suppressing street-level users who hack raw socket networks.
- **History**: Formed during Epoch 0xFF of the Corporate Monopolization, this universe split from the magitech paradigm when mages compiled the first physical network layers, which were eventually privatized by Secura Corp.
- **Why It Exists**: To test concurrency, processes, multi-threading, networking protocols, and basic offensive/defensive security.
- **How Reality Works**: Concrete, neon lights, and city systems are instances of corporate software. Hackers use portable cyber-decks to establish WebSocket connections to these systems, injecting custom payloads into active process buffers.

## 2. Aesthetic & Visual Identity
- **Visual Identity**: Neon hot-pink and cyan grids, horizontal scanlines, glowing cyber-decks, dark rainy streets, flickering terminal screens.
- **Architecture Style**: Brutalist megastructures covered in holographic ads and fiber-optic bundles.
- **HUD Theme**: Crt scanlines, retro-futuristic console grids, green terminal text.
- **Color Palette**: Cyber Cyan, Hotline Pink, Deep Asphalt Black.
- **Fonts**: Monospace, Retro Console.
- **Terminal Style**: Rugged, metal-plated hacker decks with glowing keys and port jacks.
- **Music Style**: Hard-driving synthwave, industrial electronic beats, digital noise sweeps.
- **Sound Effects**: Socket connection pings, keyboard clacks, digital static, humming server fans.
- **Boss Soundtrack**: High-tempo techno layered with glitchy guitar riffs and alarms.

## 3. Systems & Mechanics
- **Programming Paradigm**: Concurrency & Socket Scripting (JS/Python/Go).
- **Player Role**: Data Ghost (moves undetected through runtime systems by staying below CPU usage limits).
- **Unique Mechanics**: *Vigilance Meter* (Every code execution raises corporate AI awareness; players must optimize scripts to avoid alarms).
- **Traversal Mechanics**: *Grid Surfing* (Dashing along virtual network wires).
- **Mini Games**: *Port Scanning* (Intercepting packet routes by finding open ports).
- **Random Events**: *Firewall Scans* (Sudden sweeps that temporarily disable specific language compilers).
- **Environmental Hazards**: *Data Leakage Storms* (Floods memory buffers, adding random characters to the player's editor).
- **Puzzle Philosophy**: Bypass gate-checks by exploiting asynchronous race conditions or injecting overflow inputs.
- **Programming Concepts Taught**: Threads, processes, race conditions, synchronization, memory locks, socket routing.
- **Difficulty Curve**: Beginner to Medium. Focuses on performance and asynchronous timing.
- **Progression**: Unlocks JavaScript and Python executors in the editor panel.
- **Endgame Role**: Acts as the networking hub for connecting remote multi-server challenges.

## 4. Inhabitants & Factions
- **Mentor**: *Ada* (A veteran syndicate hacker who teaches the player memory exploits).
- **Merchant**: *Byte* (A cybernetic street vendor trading RAM chips and custom scripts).
- **Scientist**: *Dr. Turing* (A defector researcher mapping corporate network topologies).
- **Guide**: *Ping* (A lightweight ping daemon that assists with route diagnostic).
- **Comic Relief**: *Null* (A street punk whose deck is so buggy it constantly outputs null pointer errors).
- **Main Villain**: *Securitas-9* (The primary corporate firewall AI).
- **Final Boss**: *The Archon Runtime* (The system manager coordinating corporate threads).
- **Minor Bosses**: *The Port Router*, *Buffer Golem*.
- **Enemy Types**: Proxy Sentries, Memory Leakers, Port Scanners.
- **Elite Enemies**: Syslog Wardens (Heavy security bots that lock the CLI if code contains syntax errors).
- **Factions**: The Nakamoto Syndicate, Secura Corp, The Red-Division Hackers.
- **Cities**: *Neon City*, *The Cypher Slums*.
- **Regions**: *Corporate Hub 7*, *The Under-Grid*.
- **Biomes**: Metallic Canyons, Digital Sewers.
- **Landmarks**: *Nakamoto Mainframe Spire*, *The Great Firewall Gate*.

## 5. Economy & Collectibles
- **Resources**: CPU Cycles, Memory Shards.
- **Currency**: Cyber Credits.
- **Crafting Materials**: RAM Chips, Fiber Optic Connectors.
- **Collectibles**: *Zero-Day Exploit Archives* (Lore files describing corporate cover-ups).
- **Legendary Items**: *The Nakamoto Deck* (Increases operations budget by 50%).
- **Artifacts**: *The Decryption Core* (Allows bypass of binary security checks).
- **Lore Books**: *The Cypherpunk Manifesto* (A text detailing the history of the Net rebellion).

## 6. Narrative & Lore
- **Dialogue Style**: Technical, street-smart, heavily reliant on hacker slang.
- **Cutscene Ideas**: Glitchy camera panning down a dark neon alleyway as a hacker plugs a cable into a street terminal.
- **Cross-Universe Connections**: Secura Corp is currently trying to extract the *Origin BIOS keys* from the *Aether Library*.
- **Why This Universe Matters**: It introduces the player to the reality of runtime constraints, showing that code efficiency determines survival.
