# Universe 03: Cosmic Void

## 1. Core Overview
- **Overview**: An eldrich, cosmic void where reality exists as an Intermediate Representation (IR) compiled from the thoughts of the First Compiler. Players use gravity lenses to manipulate the AST (Abstract Syntax Tree) nodes of space-time, rewriting the structural equations of dimensions.
- **History**: Compiled during the original Bootstrap Epoch, the Cosmic Void is the raw pipeline layer connecting high-level logic to bare-metal hardware.
- **Why It Exists**: To teach compiler theory, Intermediate Representation, AST mutation, tree optimization, and structural graphics.
- **How Reality Works**: Matter is represented as scene graph nodes in the universal AST. The player can rewrite dimensional equations by shifting compiler optimization flags, folding gravity fields, or mutating AST edges.

## 2. Aesthetic & Visual Identity
- **Visual Identity**: Swirling black starlight, gravitational lensing warp ripples, glowing AST wireframes, violet-black void.
- **Architecture Style**: Floating crystalline structures connected by glowing light vectors.
- **HUD Theme**: Geometric wireframes, vector graphs, spectral analysis charts.
- **Color Palette**: Void Violet, Starlight White, Deep Obsidian.
- **Fonts**: Thin Vector, Spectral Monospace.
- **Terminal Style**: Levitating crystalline obelisks that project holographic AST trees.
- **Music Style**: Eerie ambient choral pads, distant space-echoes, deep sub-bass vibrations.
- **Sound Effects**: Glass chimes, gravitational hums, echoey compiler log reports.
- **Boss Soundtrack**: Dark orchestral cybernetic movements with massive choir swells and clockwork ticking.

## 3. Systems & Mechanics
- **Programming Paradigm**: Compiler & AST Manipulation (LLVM IR / LISP-like AST).
- **Player Role**: Geometry Weaver (manipulates spatial matrices by changing AST properties).
- **Unique Mechanics**: *Thread Shunting* (Swapping active execution threads asynchronously to bypass locks).
- **Traversal Mechanics**: *Gravity Warp* (Folding dimensions to instantly teleport between nodes).
- **Mini Games**: *AST Pruning* (Deleting dead branches from code loops to save optimization cycles).
- **Random Events**: *Decoherence Ripples* (Sudden fluctuations that scramble memory mappings).
- **Environmental Hazards**: *Black Hole Collectors* (Garbage collectors that delete variables if not kept alive).
- **Puzzle Philosophy**: Modify the structural scene graph of the universe to bypass physical obstacles.
- **Programming Concepts Taught**: AST parsing, Intermediate Representation, tree balancing, memory deallocation, 3D matrices.
- **Difficulty Curve**: Medium. Requires abstract spatial thinking and compiler logic.
- **Progression**: Unlocks compiler-optimization toolsets in the editor.
- **Endgame Role**: Acts as the compilation compiler layer, optimizing player scripts before they target other worlds.

- **Cross-Universe Connections**: The Void Watchers are currently trying to prevent the *Temporal Wardens* from freezing history logs.
- **Why This Universe Matters**: It teaches that all high-level logic must compile down to intermediate forms before acting on the world.
