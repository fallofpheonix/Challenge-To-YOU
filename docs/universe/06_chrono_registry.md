# Universe 06: Chrono Registry

## 1. Core Overview
- **Overview**: A clockwork world of brass gears, pendulum towers, and infinite copper tape registry reels. In this temporal database, history is tracked as git-commits. Players must navigate commit logs, rebase timelines, and resolve merge conflicts to correct historical anomalies.
- **History**: Offloaded during Epoch 0x0A of the Great Fork, this universe was constructed to manage state synchronization across the multiverse partitions, preventing time-based race conditions.
- **Why It Exists**: To teach version control systems (VCS), Git commands (commit, rebase, merge, cherry-pick), merge conflict resolution, and transactional rollback databases.
- **How Reality Works**: Time is a repository. Events are commits. Changing the past requires cherry-picking clean commits onto old branches or force-pushing a corrected timeline.

## 2. Aesthetic & Visual Identity
- **Visual Identity**: Brass clockwork overlays, ticking pendulum clocks, infinite tape rolls, copper gear wheels, emerald timelines.
- **Architecture Style**: Victorian Steam-Punk meets database server room. Large brass arches with spinning gear wheels and tape reels.
- **HUD Theme**: Commit graphs, diff block displays, timeline tracks.
- **Color Palette**: Chrono Brass, Timeline Emerald, Copper Brown.
- **Fonts**: Monospace Segmented, Victorian Serif.
- **Terminal Style**: Ornate brass cases with typewriter keys, rotating dials, and paper tape printer slots.
- **Music Style**: Ticking metronomes layered with classical string arrangements, cellos, and low-frequency pulses.
- **Sound Effects**: Typewriter keys, ticking clocks, paper printing rolls, gear clicks.
- **Boss Soundtrack**: Intense clockwork ticking rhythms, orchestral bells, and dramatic string build-ups.

## 3. Systems & Mechanics
- **Programming Paradigm**: Git Version Control & Transactional Queries (Git CLI commands).
- **Player Role**: Time Rebaser (resolves timeline merge conflicts to bypass temporal blockades).
- **Unique Mechanics**: *Rebase* (Reordering past events on a branch to prevent conflict collisions).
- **Traversal Mechanics**: *Timeline Leap* (Sprinting along parallel commit lines).
- **Mini Games**: *Diff Matching* (Selecting corresponding code lines to resolve conflicts).
- **Random Events**: *Merge Conflicts* (Sudden intersections that freeze active processes).
- **Environmental Hazards**: *Detached HEAD Storms* (Temporarily disconnects the player's editor from the active level branch).
- **Puzzle Philosophy**: Edit past state logs using Git commands to alter current system permissions.
- **Programming Concepts Taught**: Commits, branches, merging, rebasing, cherry-picking, diffing, repository history.
- **Difficulty Curve**: Medium. Requires understanding git graphs and conflict resolution.
- **Progression**: Unlocks temporal rollback commands in the CLI.
- **Endgame Role**: Acts as the backup recovery matrix, allowing players to reload saved profile states after a system purge.

## 4. Inhabitants & Factions
- **Mentor**: *Chronos* (A database administrator defected from the Registry, teaching git rebases).
- **Merchant**: *Penny* (A clockwork merchant selling commit tickets and timeline upgrades).
- **Scientist**: *Dr. Lovelace* (A researcher charting repository timelines).
- **Guide**: *HEAD* (A small floating pointer pointing to the active commit node).
- **Comic Relief**: *Diffy* (A glitchy robot that constantly prints conflicting text lines).
- **Main Villain**: *Delta-T* (The Grand Timeline Auditor locking branches).
- **Final Boss**: *The Eternal Merge* (The massive commit manager governing history).
- **Minor Bosses**: *The Conflict Golem*, *Branch Master*.
- **Enemy Types**: Pipeline Jammers, Conflict Specters, Branch Corruptors.
- **Elite Enemies**: Merge Enforcers (Heavy constructs that block timeline transitions until diffs are resolved).
- **Factions**: The Temporal Wardens, The Git Rebels.
- **Cities**: *Greenwich Sector*, *Rebase Ville*.
- **Regions**: *The Repository Valleys*, *The Clockwork Spans*.
- **Biomes**: Gear swamps, Paper Tape dunes.
- **Landmarks**: *The Great Pendulum Spire*, *The Commit Gateway*.

## 5. Economy & Collectibles
- **Resources**: Commit Stamps, Copper Tape.
- **Currency**: Chrono Credits.
- **Crafting Materials**: Clockwork Gears, Brass Plates.
- **Collectibles**: *Historical Commit Logs* (Files detailing the birth of the Registry).
- **Legendary Items**: *The Git Quill* (Allows modifying commit comments).
- **Artifacts**: *The Rebase Key* (Allows merging branches without resolving trivial conflicts).
- **Lore Books**: *De Tempore et Commit* (An ancient git guide).

## 6. Narrative & Lore
- **Dialogue Style**: Stately, formal, heavily reliant on version control terms.
- **Cutscene Ideas**: Giant clockwork gears turning, pulling a strip of paper tape covered in binary numbers through a puncher.
- **Cross-Universe Connections**: Delta-T is currently tracking the *Void Watchers* to freeze their asynchronous threads.
- **Why This Universe Matters**: It teaches that tracking state history is crucial for maintaining integrity and recovering from failures.
