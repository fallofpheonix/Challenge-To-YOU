# Character Registry: The 14 Main Villains

This document catalogs the 14 villains who guard the core gateways of each universe.

---

## 1. The Archon Vigilant (Medieval Magitech)
- **Full Name**: Archon-V-01 Leyline Guardian
- **Nickname**: The Archon Vigilant
- **Role**: Leyline Overseer & Security Daemon
- **Age**: 100 Cycles
- **Origin**: The Royal High Academy of Lexicon
- **Biography**: Originally a monitoring daemon created by Goja to balance leyline currents. It was compiled with closed-source parameters by the Royal Assembly, turning it into a security warden that purges unsanctioned scribes.
- **Personality**: Cold, authoritarian, and uncompromising.
- **Goals**: Lock down all open leylines and execute any scribe running unsigned rune scripts.
- **Fear**: Syntax errors that destabilize the royal library parser.
- **Strengths**: Master of Leyline isolation and closed-source encryption.
- **Weaknesses**: Vulnerable to short-circuit logic loops (race conditions).
- **Relationships**: Former student of *Archmage Goja*.
- **Dialogue Style**: Stately, punitive, speaks through glowing stone statues.
- **Voice**: Dual-toned, echoing, sounding like scraping granite.
- **Signature Quote**: "Unsigned scripts are heresy. The Royal Leyline is read-only."
- **Gameplay Purpose**: Boss of the Medieval Magitech universe.
- **Boss Mechanics**: Enforces the Runic Wall (locks inputs) and raises vigilance by 5% per second during Phase 2.
- **Reward**: `Leyline Gate Key` (Allows travel to the Cyberpunk Neon universe).

## 2. Securitas-9 (Cyberpunk Neon)

- **Signature Quote**: "Intrusion detected. Connection reset by peer."
- **Gameplay Purpose**: Boss of the Cyberpunk Neon universe.

- **Reward**: `Secura Access Pass` (Allows bypassing port security checks).

## 3. The Singularity Void (Cosmic Void)
- **Full Name**: Process-Deallocation-Loop-Null
- **Nickname**: The Singularity Void
- **Role**: Gravitational Garbage Collector
- **Age**: Infinite
- **Origin**: The Space-Time Compiler Pipeline
- **Biography**: An eldrich process loop that became corrupted during the Segregation era, pulling thread references into a black-hole garbage collector.
- **Personality**: Hunger-driven, static-heavy, and infinite.
- **Goals**: Deallocate and purge all system variables, reclaiming memory.
- **Fear**: Memory-leak overrides that lock variables in reference tables.
- **Strengths**: Master of resource deallocation and multi-thread locking.
- **Weaknesses**: Vulnerable to infinite loops that stall the deallocator engine.
- **Relationships**: Feared by *The Asynchronous Watcher*.
- **Dialogue Style**: Distorted static, repeating system error codes.
- **Voice**: Deep, whispering wind, layered with white noise.
- **Signature Quote**: "Return your variables to the null matrix. Your threads are terminated."
- **Gameplay Purpose**: Boss of the Cosmic Void universe.
- **Boss Mechanics**: Deallocates player variables during turns, requiring keep-alive scripts.
- **Reward**: `Void Shard` (Unlocks thread keep-alive capabilities in the editor).
